📐 LEDVision Setup Guide

  1. Connect the ColorLight card to a Windows PC running LEDVision.
  2. Open Controller → Detect Receiver Cards.
  3. Create a screen. Set total resolution to match the canvas size shown in the preview.
  4. For each port, set panel type, scan type (usually 1/16 for P5), and the X offset matching Canvas X above.
  5. Click Send to Receiver to save permanently.
  6. Disconnect from Windows. Connect USB-GigE to Pi.
